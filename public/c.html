C Programming - Systems Programming Language

C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs in the early 1970s. It was designed to be compiled using a relatively straightforward compiler, to provide low-level access to memory, to provide language constructs that map efficiently to machine instructions, and to require minimal run-time support.

Key Features

  • Low-level Access: Direct memory manipulation with pointers
  • Portable: Can be compiled on various platforms
  • Efficient: Close to assembly language performance
  • Structured: Supports structured programming constructs
  • Standard Library: Rich set of built-in functions

Basic C Program

#include int main() { printf("Hello, World!\n"); // Variables and data types int number = 42; float pi = 3.14159; char letter = 'A'; // Arrays int numbers[5] = {1, 2, 3, 4, 5}; // Loops for (int i = 0; i < 5; i++) { printf("Number: %d\n", numbers[i]); } return 0; }

Pointers and Memory Management

#include #include int main() { // Pointer declaration int *ptr; int value = 100; // Assigning address to pointer ptr = &value; // Accessing value through pointer printf("Value: %d\n", value); printf("Address: %p\n", (void*)ptr); printf("Value through pointer: %d\n", *ptr); // Dynamic memory allocation int *dynamicArray = (int*)malloc(5 * sizeof(int)); if (dynamicArray != NULL) { // Use the allocated memory for (int i = 0; i < 5; i++) { dynamicArray[i] = i * 10; } // Free allocated memory free(dynamicArray); } return 0; }

Structures and Functions

#include #include // Structure definition struct Person { char name[50]; int age; float height; }; // Function declaration void printPerson(struct Person p); struct Person createPerson(char name[], int age, float height); int main() { // Creating structure instances struct Person person1 = createPerson("Alice", 25, 165.5); struct Person person2 = createPerson("Bob", 30, 180.0); // Using functions printPerson(person1); printPerson(person2); return 0; } // Function implementation struct Person createPerson(char name[], int age, float height) { struct Person p; strcpy(p.name, name); p.age = age; p.height = height; return p; } void printPerson(struct Person p) { printf("Name: %s, Age: %d, Height: %.1f cm\n", p.name, p.age, p.height); }

File Handling

#include int main() { FILE *file; char buffer[100]; // Writing to file file = fopen("data.txt", "w"); if (file != NULL) { fprintf(file, "Hello, C Programming!\n"); fprintf(file, "This is a test file.\n"); fclose(file); } // Reading from file file = fopen("data.txt", "r"); if (file != NULL) { while (fgets(buffer, sizeof(buffer), file) != NULL) { printf("%s", buffer); } fclose(file); } return 0; }

Getting Started

To start with C programming:

  1. Install a C compiler (GCC, Clang, or MSVC)
  2. Use a text editor or IDE (VS Code, Dev-C++, Code::Blocks)
  3. Learn basic syntax and data types
  4. Understand pointers and memory management
  5. Practice with small programs and algorithms

Common Use Cases

  • System Programming: Operating systems, device drivers
  • Embedded Systems: Microcontrollers, IoT devices
  • Game Development: Performance-critical applications
  • Scientific Computing: Numerical analysis, simulations
  • Network Programming: Low-level network protocols

Best Practices

  • Always check return values from functions
  • Free dynamically allocated memory
  • Use meaningful variable and function names
  • Include proper error handling
  • Follow consistent coding style
