C++ Programming - Object-Oriented Systems Language

C++ is a powerful general-purpose programming language created by Bjarne Stroustrup as an extension to the C programming language. It offers a perfect blend of high-level and low-level programming features, making it ideal for systems programming, game development, and applications requiring high performance and low-level memory manipulation.

Key Features

  • Object-Oriented: Classes, inheritance, polymorphism, encapsulation
  • Generic Programming: Templates for type-independent code
  • Memory Management: Both automatic and manual memory control
  • Performance: Close to C performance with high-level abstractions
  • Standard Template Library: Rich collection of containers and algorithms

Basic C++ Program

#include #include using namespace std; int main() { cout << "Hello, C++ World!" << endl; // Variables and data types int number = 42; double pi = 3.14159; string message = "C++ is powerful!"; // Arrays int numbers[] = {1, 2, 3, 4, 5}; // Range-based for loop (C++11) for (int num : numbers) { cout << "Number: " << num << endl; } return 0; }

Classes and Objects

#include #include using namespace std; class Rectangle { private: double width; double height; public: // Constructor Rectangle(double w, double h) : width(w), height(h) {} // Member functions double getArea() const { return width * height; } double getPerimeter() const { return 2 * (width + height); } // Getters and setters double getWidth() const { return width; } double getHeight() const { return height; } void setWidth(double w) { width = w; } void setHeight(double h) { height = h; } }; int main() { Rectangle rect(5.0, 3.0); cout << "Area: " << rect.getArea() << endl; cout << "Perimeter: " << rect.getPerimeter() << endl; return 0; }

Templates and Generic Programming

#include using namespace std; // Function template template T maximum(T a, T b) { return (a > b) ? a : b; } // Class template template class Stack { private: T* elements; int capacity; int top; public: Stack(int size) : capacity(size), top(-1) { elements = new T[capacity]; } ~Stack() { delete[] elements; } void push(T value) { if (top < capacity - 1) { elements[++top] = value; } } T pop() { if (top >= 0) { return elements[top--]; } return T(); } bool isEmpty() const { return top == -1; } }; int main() { // Using function template cout << "Max of 5 and 10: " << maximum(5, 10) << endl; cout << "Max of 3.14 and 2.71: " << maximum(3.14, 2.71) << endl; // Using class template Stack intStack(5); intStack.push(1); intStack.push(2); intStack.push(3); while (!intStack.isEmpty()) { cout << "Popped: " << intStack.pop() << endl; } return 0; }

STL Containers and Algorithms

#include #include #include #include #include using namespace std; int main() { // Vector - dynamic array vector numbers = {3, 1, 4, 1, 5, 9, 2, 6}; // Sorting sort(numbers.begin(), numbers.end()); // Finding elements auto it = find(numbers.begin(), numbers.end(), 5); if (it != numbers.end()) { cout << "Found 5 at position: " << (it - numbers.begin()) << endl; } // Map - key-value pairs map ages; ages["Alice"] = 25; ages["Bob"] = 30; ages["Charlie"] = 35; // Iterating through map for (const auto& pair : ages) { cout << pair.first << ": " << pair.second << " years" << endl; } // Lambda functions auto isEven = [](int n) { return n % 2 == 0; }; // Count even numbers int evenCount = count_if(numbers.begin(), numbers.end(), isEven); cout << "Even numbers count: " << evenCount << endl; return 0; }

Modern C++ Features (C++11 and beyond)

#include #include #include using namespace std; int main() { // Auto keyword for type inference auto message = "Hello, Modern C++!"; auto numbers = {1, 2, 3, 4, 5}; // Smart pointers unique_ptr ptr1 = make_unique(42); shared_ptr ptr2 = make_shared(100); // Lambda expressions vector vec = {1, 2, 3, 4, 5}; for_each(vec.begin(), vec.end(), [](int& n) { n *= 2; }); // Range-based for loop for (const auto& num : vec) { cout << num << " "; } cout << endl; // nullptr instead of NULL int* ptr = nullptr; return 0; }

Getting Started

To start with C++ programming:

  1. Install a C++ compiler (GCC, Clang, or MSVC)
  2. Use an IDE (Visual Studio, CLion, Code::Blocks)
  3. Learn basic syntax and OOP concepts
  4. Master templates and STL
  5. Practice with modern C++ features

Common Use Cases

  • Game Development: Unreal Engine, game engines
  • System Software: Operating systems, drivers
  • High-Performance Applications: Trading systems, simulations
  • Embedded Systems: IoT devices, microcontrollers
  • Desktop Applications: Qt applications, system tools

Best Practices

  • Use RAII (Resource Acquisition Is Initialization)
  • Prefer smart pointers over raw pointers
  • Use const correctness
  • Follow the Rule of Three/Five
  • Use modern C++ features when possible
